# SpellBack

> 把你遇到的生词，真正拼出来。

SpellBack 是一个完全本地运行的私人单词复习工具。它不提供庞大的课程和预制词书，而是专注于一件事：帮助你记住自己在阅读、工作和学习中真正遇到的生词。

## 直接使用：只需要一个文件

普通用户不需要安装软件、不需要注册账号，也不需要了解 GitHub：

1. 获取 `SpellBack.html`。
2. 把它放在一个固定文件夹中。
3. 双击文件，用 Chrome、Edge 或 Firefox 打开。
4. 开始添加和复习单词。

你可以直接通过微信、QQ、邮件或 U 盘把这个 HTML 文件发给其他人。页面样式、图标和程序逻辑全部包含在文件内部，断网也能完整使用。

复习时只显示中文，你必须拼出英文。完成后重新评估 1–5 级熟练度，程序再按你的规则安排下一次复习。

## 为什么再做一个背单词工具？

成熟的背词产品擅长提供海量词表、图片场景、权威释义、考试内容和完整课程。SpellBack 不复制这条路线，而是选择一个更小、更个人的方向：

| 关注点 | SpellBack 的选择 |
| --- | --- |
| 学什么 | 只收录你亲手添加、真正需要记住的词 |
| 怎么复习 | 固定采用“中文提示 → 英文拼写”的主动回忆 |
| 谁决定节奏 | 你可以修改五级间隔、每日上限和错词策略 |
| 如何判定 | 支持严格/宽松模式，以及每个词的多个可接受拼写 |
| 数据归谁 | 不注册、不联网；词库保存在本地并可导出为 JSON |
| 产品干扰 | 没有广告、签到、排行榜和课程推荐 |

它不是另一个英语内容平台，而是一张安静、透明、可掌控的私人复习桌。

## 核心功能

- **主动拼写**：根据中文输入英文，避免“看着眼熟”带来的记忆错觉。
- **五级熟练度**：默认间隔为 1、2、4、7、15 天，每一级都可以修改。
- **每日复习上限**：控制每天的负担，未完成的到期词保留到下一轮。
- **错词次日重现**：拼错的词可以强制在第二天再次出现。
- **易错词优先**：同一天到期时，优先安排累计错误更多的词。
- **可接受拼写**：为同一个词添加英美拼写、复数或其他有效答案。
- **严格/宽松判定**：宽松模式会额外忽略空格、连字符和标点。
- **标签管理**：按“阅读”“工作”“雅思”等个人场景整理和筛选。
- **正确率记录**：在词库中查看每个词的实际拼写正确率。
- **本地数据**：支持 JSON 导入导出，方便备份、迁移和自行处理。

## 开发者使用

源码没有第三方依赖或后端服务。开发时直接打开 `index.html`；发布前生成单文件版本：

```bash
node build_single_html.js
```

构建结果是 `SpellBack.html`。Windows 下也可以双击 `open_vocab_web.bat` 打开发行版。

## 复习流程

```text
添加英文、中文、标签和初始熟练度
                    ↓
            单词进入到期队列
                    ↓
              显示中文释义
                    ↓
              手动拼写英文
                    ↓
        检查答案并更新 1–5 级
                    ↓
          按个性化间隔安排下次复习
```

默认调度：

| 熟练度 | 含义 | 默认间隔 |
| --- | --- | ---: |
| 1 | 陌生 | 1 天 |
| 2 | 眼熟 | 2 天 |
| 3 | 一般 | 4 天 |
| 4 | 熟悉 | 7 天 |
| 5 | 掌握 | 15 天 |

这些数字只是默认值。你可以在“设置”页面把它们改成适合自己的节奏。

## 个性化示例

### 准备考试

- 每日上限设为 80。
- 五级间隔设为 `1 / 2 / 3 / 5 / 10`。
- 开启错词次日重现和易错词优先。
- 用标签区分 `四级`、`六级`、`雅思`。

### 日常阅读

- 每日上限设为 15。
- 五级间隔设为 `1 / 3 / 7 / 14 / 30`。
- 为 `colour / color` 等写法添加可接受拼写。
- 用标签记录词语来源，例如 `论文`、`小说`、`工作邮件`。

## 数据与隐私

- 所有数据默认保存在当前浏览器的 `localStorage` 中。
- 项目不会上传词库、统计或使用记录。
- 不需要账号，也没有远程分析脚本。
- 清理浏览器数据，或移动、重命名 `SpellBack.html`，可能产生新的本地存储空间，请定期在“设置”中导出 JSON 备份。
- 导出的文件是普通 JSON，可以使用文本编辑器或脚本读取。
